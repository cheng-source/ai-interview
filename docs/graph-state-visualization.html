4 三种 Reducer 行为

🔁 替换 replace (默认)

currentStage · candidate · position · techRound · behavioralRound · qaCount · candidateAnswer · resumeText · candidateIntro · interviewType
currentStage 'icebreaker' → 'technical' 'icebreaker' → 'qa' 'icebreaker' → 'done' 新值直接覆盖旧值

➕ 追加 append

messages · answerHistory
messages [msg1, msg2] + [msg3] = [msg1, msg2, msg3] 新条目追加到数组末尾 永不删除已有内容

🔀 合并 merge

scores
scores {tech:0, behav:0, overall:0} + {tech:7} = {tech:7, behav:0, overall:0} 后续再 + {behav:8} = {tech:7, behav:8, overall:0} 新 key 覆盖旧 key 未提供的 key 保留原值
