System Architecture Overview

NestJS + LangGraph JS + Vue3 — 全 TypeScript 技术栈

Layered Architecture

Layer 1 — 前端展示

面试界面 · 语音输入(可选) · 实时代码编辑器 · 结果报告

Vue3 SPA (HR管理端)

岗位管理 · 面试配置 · 报告查看 · 数据看板

WebSocket / REST API
Layer 2 — NestJS 后端服务
Gateway / Controller
WebSocket 实时通信 · REST API · 鉴权中间件
Service Layer
面试管理 · 题库服务 · 评分服务 · 报告生成 · 语音TTS
Prisma ORM
PostgreSQL · Redis (Session) · 文件存储

Key Design Decisions

Why This Works

  • 全 TypeScript — NestJS 直接 import @langchain/langgraph，零跨语言开销
  • LangGraph 在 NestJS Service 内运行，复用 DI 容器和数据库连接
  • WebSocket 直连，LLM token 流式推送到前端
  • 单进程部署，调试和测试简单
